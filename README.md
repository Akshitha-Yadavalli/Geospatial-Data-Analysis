# Geospatial Data Analysis

## 📌 Project Overview

This project analyzes geographical sales and demand data to identify optimal locations for business expansion.

Using location-based analytics and mapping tools, the project visualizes sales performance and highlights regions with high demand but low business presence.

---

## 🎯 Objectives

- Analyze sales distribution across locations
- Identify regions with high demand
- Detect underserved markets
- Recommend business expansion opportunities
- Visualize data using interactive maps

---

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- Folium
- NumPy

---

## 📈 Key Findings

Cities with:

- High Demand Score (>90)
- Low Existing Stores (≤1)

are identified as potential business expansion opportunities.

Examples:

- Vijayawada
- Surat
- Indore
- Pune
- Visakhapatnam

- Open new stores in underserved high-demand cities.
